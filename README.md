# 株式指標トラッカー

決算資料などから集めた **株価 / EPS / BPS / PER / PBR** を、銘柄ごとに時系列で記録するシンプルなWebアプリです。

## できること

- 銘柄コードと銘柄名を登録
- 日付ごとの指標記録（PER/PBRは未入力なら自動計算）
- 銘柄で絞り込んだ時系列テーブル表示
- 指標を切り替えられる折れ線グラフ
- データのJSONエクスポート / インポート
- （任意）画像OCRでフォーム補助入力

## 使い方

1. `index.html` をブラウザで開く
2. 「銘柄登録」で銘柄を追加
3. 「指標を記録」からデータを登録
4. 「記録一覧（時系列）」で推移を確認

> データはブラウザの `localStorage` に保存されます。

## ローカルサーバーで確認する場合

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開いてください。

# 株式指標トラッカー

日次で株価を記録し、選択したEPSモードに基づいて **PER（株価 ÷ EPS）** を自動算出するWebアプリです。

## 設計思想

- EPSは「会計データ / 期待値」で、日次では頻繁に変わらない
- 株価は日次で変動する
- PERは市場評価として、**株価変動に応じて日次で変動**する

## EPSモード

1. **固定EPS（FIXED_EPS）**
   - 1つのEPSを固定値として利用
2. **四半期EPS（QUARTERLY_EPS）**
   - 四半期EPSを登録し、直近4四半期合計でTTM EPSを使用
3. **予想EPS（FORWARD_EPS）**
   - 将来予想EPSを固定値として利用

> EPSを変更すると、保存済みの日次PERは再計算されます。

## 主な機能

- 銘柄登録
- EPSモードごとの管理
- 日次株価入力時のPER即時計算
- 日付・株価・PERの一覧表示
- PER推移グラフ
- JSONエクスポート / インポート
- 画像OCRによる株価補助入力（任意）

## 起動

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開いてください。
